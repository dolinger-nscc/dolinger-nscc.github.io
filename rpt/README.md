# Covid-19 Cases

This report is an ongoing analysis of the Covid-19 response primarily for Canada. Data for the United States will also be included for a compare / contrast to identify effectiveness of distancing intervention.  
The report will leverage [Flourish Studio](https://flourish.studio/). 
 
## Assumptions
The following is accepted as fact and may change as more information about the virus is established.
- Mortality rates, infection rates, etc. for the Covid-19 virus have not been universally established.

## Assumptions
This report makes the following assumptions and may change from time to time during report updates. Because of limited trace testing currently available the following assumption will be used in analysis and projections.
- Between 25 percent to 50 percent of people who are infected are asymptomatic.

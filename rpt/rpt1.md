# Current Cases by Province

This report will be kept current to track cases "at a glance" for each province. The report reflects the current case count and a quick percentage of active vs recovered for the reported cases. 
The bullet chart (Daily Confirmed Cases Since 1 Mar) gives a trend visualisation of daily cases reported for the province since 1 March 2020.
